## About Me

I'm a Cyber Security undergraduate pursuing a B.Tech in Computer Science and Engineering with a specialization in Cyber Security at CMR College of Engineering and Technology (Expected Graduation: June 2027, CGPA: 8.19/10). My focus areas are ethical hacking, bug bounty hunting, and offensive security research.

I actively work on identifying and responsibly disclosing web application vulnerabilities — including authentication flaws, access control issues, security misconfigurations, and business logic flaws. I regularly practice through CTFs, hands-on labs, and real-world simulations, running native Kali Linux as my primary operating system.

My current interests span Penetration Testing, Malware Analysis, Red Teaming, and Security Research, and I'm continuously building practical experience through independent labs and research projects.
